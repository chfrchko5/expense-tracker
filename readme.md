# Expense Tracker

an application built in python used to track and save expenses

# Description
simple app that can be used to track various expenses  
can add, list, delete expenses  
summary option to summarize all expenses or expense for a specific month


## Installation

pull the repo, install requirements.txt and use the "tracker.py" located in 'src/expense-tracker/' directory

## Usage

```python
# list help options to start using the application
tracker.py --help
```
